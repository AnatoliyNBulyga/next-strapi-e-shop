## Full-stack e-commerce app based on Next.js and Strapi CMS

**Stack:**
- Next.js
- Typescript.js
- PostgreSQL
- Tailwindcss
- React hook form
- Zustand
- Strapi headless CMS
- Stripe
- Strapi build-in auth api

**You can evaluate the work of the app here:** https://next-strapi-e-shop-client.vercel.app/

## How to run

1. cd api

2. npm install

3. npm run dev

4. Open a new terminal

5. cd client

6. npm install

7. npm run dev
